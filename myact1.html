Career Opportunities after Graduation

Software Developer

A software developer is a professional who designs, creates, tests, and maintains software applications or systems.

Data Scientist

A data scientist is a professional who uses scientific methods, algorithms, and systems to extract insights and knowledge.

Web Developer

Web developer is a professional who builds and maintains websites and web applications, focusing on both the technical functionality and user experience.

Database Administrator

Database Administrator (DBA) manages, secures, and maintains an organization's databases, ensuring their performance, availability, and integrity.

Game Developer

A game developer is a professional who designs, creates, and programs video games, working on various aspects such as gameplay mechanics,.

UI/UX Designer

A UI/UX Designer focuses on creating user interfaces (UI) and enhancing user experiences (UX) for digital products, ensuring that they are visually appealing.
